Testing plugin for Themeisle products.

== Description ==

The plugin contains the following features:

- SDK datetime hook override via WP filters.
- SDK Black Friday options via WP filters.
- Changes the license data for products with SDK integration.
- Changing the install date of the products.
- Changing the options via WP filter for Formbricks.
